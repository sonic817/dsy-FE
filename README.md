# 다율숲 (dys-FE)

다율숲 숲체험 비회원 예약 웹사이트 프론트엔드

## 기술 스택

- Next.js 15 / TypeScript / Plain CSS

## 실행

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 빌드

```bash
npm run build
npm run start
```
